
# MERN E-COMMERCE DRESS STORE  (BRAND SHOPPERS)

This is a fully functional e-commerce website for men's, women's, and kids' clothing. The application is built using the MERN stack: MongoDB, Express.js, React, and Node.js. The website supports user authentication using JWT tokens and includes features like product browsing, adding to cart, and admin panel for managing products.


## Features

- User Authentication (Login, Logout)
- Browse Products (Men, Women, Kids)
- Add to Cart
- View Related Products
- View New Collections
- Fully Responsive Design
- Admin Panel for Managing Products (Add, List)
## Tech Stack

- Frontend: React
- Backend: Express.js/ Nodejs
- Database: MongoDB
- Authentication: JWT Tokens
- Deployment: Render.com


## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
```
